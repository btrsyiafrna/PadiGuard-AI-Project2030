# PadiGuard-AI-Project2030
PadiGuard AI is an agentic, culturally sovereign AI platform developed for Project 2030: MyAI Future Hackathon (Track 1: Padi &amp; Plates) to address Malaysia’s RM78.8 billion annual food import dependency and the 25–40% yield losses faced by smallholder farmers. Built on the complete Google AI Ecosystem Stack.
